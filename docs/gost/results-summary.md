# Достигнутые результаты

## Система голосовой конференцсвязи Voice Conference

**Дата окончания работ:** 2026-07-16
**Версия:** 2.0
**Статус:** Рабочий прототип, развёрнут на сервере

---

## 1. Краткое описание

Разработана кроссплатформенная система голосовой конференцсвязи с использованием веб-технологий (HTML5/Web Audio API/WebSocket). Система не требует регистрации, обеспечивает анонимное подключение по коду комнаты и не хранит персональные данные.

**Сервер:** https://87.242.117.240:8443

---

## 2. Реализованные функции

### 2.1. Основной функционал

| Функция | Статус | Описание |

## 4. Архитектурные решения

### 4.1. Relay-модель (не P2P)

| Преимущество | Описание |
|---|---|
| NAT traversal | Не нужен TURN/STUN |
| Масштабируемость | 1 сервер → N комнат |
| Простота | Одно WS-соединение на клиента |
| Контроль | Сервер может фильтровать/маршрутизировать |

### 4.2. Отсутствие хранения данных

| Данные | Где хранятся |
|---|---|
| Аудиопоток | Только RAM, не на диске |
| Чат | Только RAM |
| Файлы | Только uploads/ (TTL 12ч) |
| Комнаты | Только RAM |
| Логи | Только systemd journal |

### 4.3. Выбор Web Audio API

| API | Использование |
|---|---|
| ScriptProcessorNode | Захват аудио с микрофона |
| AudioBufferSourceNode | Воспроизведение аудио |
| GainNode | Audio Ducking + Gain control |
| MediaStreamSource | Подключение микрофона |
