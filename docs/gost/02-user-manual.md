## 1. Назначение и область применения

Настоящее руководство описывает порядок работы с веб-приложением «Система голосовых конференций» — платформой для организации голосовых совещаний с поддержкой текстового чата и обмена файлами.

Система предназначена для:

- проведения голосовых совещаний в реальном времени;
- обмена текстовыми сообщениями между участниками конференции;
- передачи файлов участникам конференции;
- работы в режиме прослушивания без использования микрофона.

Система не требует установки клиентского программного обеспечения и работает через веб-браузер по адресу https://87.242.117.240:8443.

## 2. Системные требования

### 2.1. Аппаратные требования

| Компонент | Минимальные требования |
|---|---|
| Процессор | 1 GHz |
| Оперативная память | 512 МБ |
| Свободное место на диске | 50 МБ |
| Микрофон | не требуется (режим прослушивания) / рекомендуется для голосового участия |

### 2.2. Программные требования

| Компонент | Требование |
|---|---|
| Операционная система | Windows 7+, macOS 10.12+, Linux |
| Веб-браузер | Google Chrome 56+, Mozilla Firefox 52+, Microsoft Edge 79+, Яндекс Браузер 17+ |
| Подключение к сети Интернет | от 64 Кбит/с |

### 2.3. Дополнительные требования

- Разрешённый доступ к микрофону в браузере (для режима передачи голоса).
- Доступ по протоколу HTTPS (порт 8443) к адресу сервера.
- Поддержка JavaScript и WebSockets в браузере.

## 3. Порядок установки и настройки

### 3.1. Установка

Установка клиентского программного обеспечения не требуется. Приложение является веб-приложением и загружается из браузера.

### 3.2. Настройка браузера

1. Откройте веб-браузер.
2. Перейдите по адресу: https://87.242.117.240:8443
3. При первом входе браузер может запросить разрешение на использование микрофона. Нажмите **«Разрешить»**, если планируете участвовать в конференции голосом.

### 3.3. Разрешение на микрофон

При первом входе в конференцию с активным микрофоном браузер отобразит диалоговое окно с запросом доступа к микрофону. Необходимо выбрать нужный микрофон из списка и подтвердить разрешение.

Если разрешение было отклонено ранее, его можно восстановить:

- **Chrome**: Настройки → Конфиденциальность → Настройки сайтов → Микрофон → добавить исключение для `https://87.242.117.240:8443`.
- **Firefox**: Адресная строка → значок замка → Параметры → Микрофон → «Разрешить».
- **Яндекс Браузер**: аналогично Chrome.

## 4. Порядок работы

### 4.1. Вход в лобби

1. Перейдите по адресу https://87.242.117.240:8443.
2. Откроется экран лобби с полями ввода и кнопкой входа.

### 4.2. Присоединение к комнате

1. В поле **«Код комнаты»** введите PIN-код. Это может быть любое число или слово (например, `1234`, `совещание`, `demo`).
2. В поле **«Время (мин)»** укажите продолжительность комнаты в минутах (от 1 до 720). По умолчанию значение определяется настройками сервера.
3. Нажмите кнопку **«Войти»**.
4. Система подключит вас к конференции. Если микрофон не запрошен ранее, браузер запросит разрешение на его использование.

### 4.3. Участие в конференции

1. После входа в комнату вы увидите заголовок со статусом, индикатором качества и таймером обратного отсчёта.
2. В области просмотра отображается список участников.
3. Панель чата находится в правой части экрана.
4. Управление голосом, файлами и выходом расположено в нижней трети экрана.

### 4.4. Выход из конференции

Нажмите кнопку **«Выйти»** в правой части нижней панели управления. При выходе соединение будет корректно закрыто.

### 4.5. Окончание времени комнаты

Когда установленное время истекает, комната автоматически закрывается. Все участники отключаются, и возвращается экран лобби.

## 5. Описание интерфейса

### 5.1. Экран лобби

Лобби — начальный экран приложения, отображаемый до входа в конференцию.

| Элемент | Описание |
|---|---|
| **Поле ввода кода комнаты** | Текстовое поле для ввода PIN-кода комнаты. Принимает любые символы — числа, буквы, спецсимволы. |
| **Поле ввода времени** | Числовое поле для указания длительности комнаты в минутах. Диапазон значений: от 1 до 720. |
| **Кнопка «Войти»** | Кнопка для входа в комнату. Активна при заполнении обязательных полей. |

### 5.2. Заголовок комнаты

Отображается в верхней части экрана после входа в конференцию.

| Элемент | Описание |
|---|---|
| **Статус** | Текущее состояние подключения (подключён, переподключение, отключён). |
| **Индикатор качества** | Числовое значение от 1 до 10, отражающее качество текущего соединения. |
| **Таймер** | Отображает оставшееся время жизни комнаты в формате минуты:секунды. |

### 5.3. Область просмотра участников

Основная область в левой части экрана. Содержит список всех участников текущей конференции.

| Элемент | Описание |
|---|---|
| **Список участников** | Отображает идентификатор каждого участника и статус его микрофона (активен / отключён / не подключён). |

### 5.4. Панель чата

Находится в правой части экрана.

| Элемент | Описание |
|---|---|
| **Область сообщений** | Прокручиваемый список текстовых сообщений участников с указанием автора и времени. |
| **Поле ввода сообщения** | Текстовое поле для набора сообщения. Отправка — по клавише Enter или кнопке отправки. |

### 5.5. Панель управления (нижняя треть экрана)

Расположена в нижней трети экрана. Элементы управления широко расставлены для предотвращения случайных нажатий.

| Элемент | Расположение | Описание |
|---|---|---|
| **Кнопка микрофона** | Слева | Включение / выключение микрофона. При включении микрофон активен, при выключении — в режиме прослушивания. |
| **Кнопка файлов** | По центру | Открытие диалога выбора файла для загрузки в конференцию. |
| **Кнопка «Выйти»** | Справа | Выход из текущей конференции и возврат в лобби. |

## 6. Работа с голосом

### 6.1. Активация микрофона

1. Нажмите кнопку микрофона (левая кнопка в нижней панели).
2. Браузер запросит разрешение на доступ к микрофону (при первом входе).
3. Выберите нужное аудиоустройство из предложенного списка.
4. Подтвердите разрешение.
5. Кнопка микрофона изменит визуальное состояние, указывая на активность.

### 6.2. Режим прослушивания

Для участия в конференции без микрофона:

1. Не нажимайте кнопку микрофона.
2. Вы будете слышать других участников, но не сможете говорить.
3. В списке участников рядом с вашим идентификатором будет отмечено, что микрофон не подключён.

### 6.3. Техническая обработка голоса

Система выполняет следующую обработку аудиопотока:

| Параметр | Значение / Описание |
|---|---|
| **Частота дискретизации** | 8 кГц (resampling с частоты микрофона) |
| **Формат** | Int16 PCM |
| **Подавление кликов** | Затухание fade-in / fade-out по 16 отсчётов |
| **Подавление тишины** | Пиковая амплитуда < 0.02 — поток не передаётся |
| **Адаптивный буфер** | 2–8 аудиочанков в зависимости от качества сети |
| **Маршрутизация** | Микрофон → WebSocket (бинарные данные) → сервер → все участники |

### 6.4. Рекомендации по использованию

- Используйте гарнитуру для уменьшения эха.
- Располагайте микрофон на расстоянии 5–15 см от рта.
- Избегайте шумных мест при передаче голоса.
- При плохом качестве связи система автоматически увеличивает буфер, что может вызвать небольшую задержку.

## 7. Работа с чатом

### 7.1. Отправка сообщения

1. Перейдите в панель чата (правая часть экрана).
2. Введите текст сообщения в поле ввода внизу панели.
3. Нажмите клавишу **Enter** или кнопку отправки.
4. Сообщение отобразится в области сообщений с указанием вашего имени и времени отправки.

### 7.2. Получение сообщений

Все входящие сообщения отображаются в области сообщений панели чата в реальном времени. При превышении количества сообщений область прокручивается автоматически.

### 7.3. Просмотр истории

Прокрутите область сообщений вверх для просмотра предыдущих сообщений. История чата сохраняется на протяжении всей сессии конференции.

## 8. Работа с файлами

### 8.1. Загрузка файла

1. Нажмите кнопку файлов (центральная кнопка в нижней панели).
2. В открывшемся диалоге выберите файл для загрузки.
3. Подтвердите выбор.
4. Файл будет передан участникам конференции.

### 8.2. Ограничения

| Параметр | Значение |
|---|---|
| **Максимальный размер файла** | 100 МБ |
| **Поддерживаемые форматы** | все файлы, доступные через встроенный диалог браузера |

### 8.3. Получение файла

Файлы, загруженные другими участниками, доступны для скачивания через панель чата или интерфейс скачивания.

## 9. Индикатор качества связи

### 9.1. Назначение

Индикатор качества связи отображается в заголовке комнаты и показывает числовое значение от 1 до 10, отражающее текущее состояние сетевого соединения.

### 9.2. Шкала значений

| Диапазон | Значение | Описание |
|---|---|---|
| 8–10 | Отличное | Стабильное соединение, минимальная задержка, отсутствие потерь пакетов. |
| 5–7 | Хорошее | Соединение стабильно, возможна небольшая задержка. |
| 3–4 | Удовлетворительное | Заметные задержки и/или потеря отдельных аудиочанков. Буфер увеличен. |
| 1–2 | Плохое | Серьёзные проблемы соединением, значительные задержки и пропуски. Рекомендуется проверить сетевое подключение. |

### 9.3. Влияние на работу

- При снижении качества автоматически увеличивается размер адаптивного буфера (от 2 до 8 аудиочанков).
- При очень низком качестве возможны прерывания в передаче голоса.
- Качество обновляется в реальном времени.

## 10. Решение проблем

### 10.1. Браузер не запрашивает микрофон

| Причина | Решение |
|---|---|
| Разрешение ранее отклонено | Сбросьте настройки микрофона для сайта https://87.242.117.240:8443 в настройках браузера (см. раздел 3.3). |
| Используется неподходящий браузер | Переключитесь на Chrome 56+, Firefox 52+, Edge 79+ или Яндекс Браузер 17+. |

### 10.2. Нет звука от других участников

| Причина | Решение |
|---|---|
| Громкость системы слишком низкая | Проверьте уровень громкости в настройках операционной системы и браузера. |
| Звуковое устройство не выбрано | Убедитесь, что в настройках звука браузера выбраны правильные наушники/колонки. |
| Соединение разорвано | Проверьте индикатор качества и статус подключения в заголовке комнаты. |

### 10.3. Другие участники не слышат мой голос

| Причина | Решение |
|---|---|
| Микрофон выключен | Нажмите кнопку микрофона для включения. |
| Браузер не имеет доступа к микрофону | Проверьте разрешение на микрофон для сайта. |
| Подавление тишины активно | Убедитесь, что микрофон работает и в него поступает звук (пик > 0.02). |

### 10.4. Низкое качество голоса

| Причина | Решение |
|---|---|
| Медленное интернет-соединение | Проверьте скорость подключения (рекомендуется от 64 Кбит/с). |
| Высокая нагрузка на сеть | Закройте другие приложения, использующие канал. |
| Отдалённое расположение от сервера | Качество определяется задержкой; при удалённости от сервера задержка увеличивается. |

### 10.5. Файл не загружается

| Причина | Решение |
|---|---|
| Файл превышает 100 МБ | Уменьшите размер файла или разбейте на части. |
| Нестабильное соединение | Дождитесь восстановления стабильного подключения и повторите попытку. |

### 10.6. Комната закрылась раньше времени

| Причина | Решение |
|---|---|
| Истекло установленное время | Время жизни комнаты ограничено значением, указанным при входе (1–720 мин). Создайте новую комнату. |
